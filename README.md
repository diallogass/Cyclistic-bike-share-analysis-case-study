# Cyclistic-bike-share-analysis-case-study
We examined how annual members and casual riders use Cyclistic bikes differently in order to identify opportunities to convert casual riders into long-term members. After cleaning and preparing the dataset in Google Sheets, key metrics such as ride length, ride frequency, and day-of-week usage were analyzed using pivot tables and visualizations. 

## Objective
The primary objective of this analysis is to understand how **annual members** and **casual riders** use Cyclistic bikes differently, and to use these insights to recommend marketing strategies that encourage membership growth.

## Dataset Used
- <a href=“https://github.com/diallogass/Cyclistic-bike-share-analysis-case-study/blob/main/Cyclist%20Data%2C%20Marketing%20Analysis.xlsx”>Dataset<a/>  

## Data Cleaning and Preparation
To ensure the dataset was accurate and ready for analysis, the following cleaning and preparation steps were completed:

- Imported Cyclistic’s historical bike-share data into **Google Sheets**
- Separated **date** and **time** into individual columns
- Calculated **ride length** for each trip
- Created a **day-of-week** column to analyze weekday vs. weekend usage
- Removed **duplicate ride IDs** to maintain data integrity
- Organized the dataset for further descriptive analysis

## Analysis Process
After cleaning the data, descriptive analysis was performed using **formulas** and **pivot tables** in Google Sheets.

### Key analytical steps included:
- Calculating **average ride length**
- Counting the number of rides by rider type
- Identifying **trends by day of the week**
- Comparing usage behavior between:
  - **Annual members**
  - **Casual riders**
- Creating visualizations to support findings and reveal behavioral patterns
